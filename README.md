# RetailFlow-Analysis-Excel-Project
## 📌 Project Overview
This project focuses on analyzing 3 years of Fast-Moving Consumer Goods data to uncover key performance indicators (KPIs) in sales, inventory, and logistics. Using a large dataset comprising over 1,000+ entries, I have built an interactive Excel Dashboard that transforms raw data into actionable business insights.

## 🛠️ Tech Stack & Skills
* **Tool:** Microsoft Excel (Advanced)
* **Techniques:** * **Power Query:** Data cleaning and transformation.
    * **Pivot Tables:** Summarizing complex data points.
    * **Data Modeling:** Creating relationships between Sales, Stocks, and Regions.
    * **Dashboarding:** Utilizing Slicers, Timelines, and Dynamic Charts for UX.

## 🚀 Key Dashboard Features
- **Sales vs. Delivery Analysis:** Comparative view of `delivered_qty` and `units_sold` to measure fulfillment efficiency.
- **Regional Heatmap:** Performance analysis across `PL-North`, `PL-Central`, and `PL-South`.
- **Category Deep-Dive:** Tracking growth in segments like **Milk, Yogurt, and Snack Bars**.
- **Promotion Impact:** Evaluating how the `promotion_flag` correlates with a surge in sales volume.
- **Stock Management:** Visualizing `stock_available` against `delivery_days` to identify lead-time bottlenecks.

## 📈 Business Insights Included
- **Growth Trends:** Yearly and monthly sales growth patterns from Jan 2022 to Dec 2024.
- **Channel Performance:** Sales distribution across **Retail, Discount, and E-commerce** channels.
- **Pricing Strategy:** Analysis of `price_unit` impact on total units sold.
- **Inventory Alerts:** Identified periods where stock levels were insufficient to meet demand.
